Mini Quiz Application
Mini Quiz Application is a console-based quiz program built using core Java concepts. It allows users to answer a series of questions and uses different lifeline-style mechanics to make the quiz more interactive and engaging.

Technologies Used
Java (Console-based, Core Java)

Description
This Mini Quiz Application runs in the console and asks the user multiple-choice questions.

It evaluates the selected answers and maintains the score as the quiz progresses.

The game flow and features are implemented using core Java concepts such as classes, objects, control statements, and basic data structures.

Key Features
Lifeline system to help the player during tough questions.

50-50 Chance: Eliminates two incorrect options to increase the chance of choosing the right answer.

Audience Poll: Simulates audience voting to suggest the most likely correct answer.

Question Swap: Allows the user to skip the current question and replace it with a new one.

How to Run
Ensure you have Java installed (JDK).

Compile the source files:

javac Quiz.java (or your main class file)

Run the application:

java Quiz
